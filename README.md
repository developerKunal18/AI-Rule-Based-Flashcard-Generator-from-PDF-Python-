# AI-Rule-Based-Flashcard-Generator-from-PDF-Python-
A tool that:  extracts text from a PDF (lecture notes / textbook / article),  creates flashcards automatically using cloze (fill-in-the-blank) style questions,  optionally makes multiple-choice questions with distractors,  exports flashcards to JSON (easy import to Anki, Quizlet, or your own app).
